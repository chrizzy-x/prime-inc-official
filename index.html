05 — Prime Pay

One wallet. Every surface.

Same wallet. Same PIN. Same balance. USSD, app, or agent API — it's all Prime Pay.

Protocol Layers
Layer 1
USSD Gateway
Via Africa's Talking. Any GSM phone, any network. Phone number is a private routing key — never on-chain. Seed phrase split across 2 USSD screens, never transmitted via SMS.
Layer 2
Prime App
Same wallet, same PIN, same balance as USSD. 3 transfer methods: phone number, wallet address, Prime Tag.
Layer 3
Agent API
Payment primitives for AI agents on any supported chain. Programmable transfers, agent-initiated settlements, machine-to-machine payments at protocol speed.
Phase 2
FFP Native Settlement
Phase 1 covers Solana, Base, ETH, and Monad via Prime Bridge at 0.1% flat. Phase 2 removes the bridge — native settlement directly on FFP.

The Story

From Lagos, building global infrastructure.

Prime Inc started with a question: why does AI infrastructure assume you have reliable power, a stable internet connection, and a bank account? Nobody building infrastructure was building from Lagos. So Riz decided to. Bootstrapped, solo, no outside capital — every product ships because the work demands it, not because a funding round did.

Why Lagos

The world's most honest product environment.

Lagos is one of the most competitive, resource-constrained, high-velocity cities on earth. Intermittent power means Prime Gadgets exists. Variable connectivity means Prime Pay's USSD layer exists. Cost-conscious, technically sophisticated users mean every product has to be genuinely useful, not just well-designed. Building here forces honesty that a comfortable environment never would.

The Conviction

The AI-native economy needs new primitives.

The next decade doesn't need more AI wrappers on top of 2010s SaaS. It needs real primitives — agent coordination protocols, verified digital identity, permissionless financial infrastructure, and payment rails that work for both humans and machines. AgentOS is the agent layer. FFP is the settlement layer. Prime Pay is the access layer. The goal isn't to compete with Silicon Valley. The goal is to build the stack that makes the next generation of companies possible — from anywhere.
